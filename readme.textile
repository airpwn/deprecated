h1. About the PDF methods (Libraries)

h2. dompdf (default)
 * http://code.google.com/p/dompdf/
 * http://alexking.org/blog/2007/11/25/dompdf-php-pdf-generation

From http://code.google.com/p/dompdf/wiki/Installation :

    chgrp www-data ./plugins/pdfview/vendors/dompdf/lib/fonts
    chmod g+w ./plugins/pdfview/vendors/dompdf/lib/fonts

If your user is not a member of the www-data group or you do not have root privileges, you can make the directory world writable and set the sticky bit:

    chmod 1777 ./plugins/pdfview/vendors/dompdf/lib/fonts
